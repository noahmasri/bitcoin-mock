# 23C1-pequenos-rustaceans
Repo for Rust Taller De Programacion 1 FIUBA

# Compilacion 
Para correr la wallet:
```bash
cargo run --bin wallet <path de configuracion>
```

Para correr el nodo:
```bash
cargo run --bin node <path de configuracion>
```
